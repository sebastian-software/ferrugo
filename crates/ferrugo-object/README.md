# ferrugo-object

`ferrugo-object` builds on `ferrugo-syntax` to load PDF object graphs, xref data, trailers, streams, page trees, and document metadata.

## Use It For

- Loading classic and modern PDF object structures.
- Resolving indirect objects and page-tree metadata with explicit resource limits.
- Decoding supported stream filters for the native renderer.
