# ferrugo-render

`ferrugo-render` turns parsed PDF page resources and content operations into display-list items and bounded raster output.

## Use It For

- Building display lists for paths, text, images, forms, patterns, and graphics resources.
- Rasterizing supported page content into RGBA buffers with explicit memory budgets.
- Sharing renderer diagnostics and timing attribution with the native backend.
